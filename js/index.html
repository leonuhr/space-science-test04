Core Concepts

  • State (Memory) Variables like count allow the page to remember things that happen over time, rather than just what was loaded initially.
  • The Event Loop JS waits for things to happen. addEventListener tells the browser: "When a user clicks this, run my function."
  • DOM Manipulation JS can reach into the HTML structure, find elements, and change their text, styles, or classes dynamically on the fly.
  • Imperative Logic You have to write step-by-step instructions. "Find the button, change its text, make it visible, increment the counter."

// 1. State: remembering data
let count = 0;

function handleCardClick() {
  // 2. Logic: incrementing the counter
  count++;
  
  // 3. DOM Manipulation: updating the UI
  actionBtn.textContent = 'Clicked!';
  
  counterText.textContent = `Clicked ${count} times`;
}

// 4. Events: listening for user actions
actionBtn.addEventListener('click', handleCardClick);
