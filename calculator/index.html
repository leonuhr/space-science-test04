🧮 Simple Calculator

📝 Build Steps

  1. HTML: Create the button grid layout
  2. CSS: Style the buttons with colors
  3. JavaScript: Add click handlers

Step 1: HTML Buttons

Create a grid with 16 buttons (0-9, operators, clear, equals)

<!-- Basic grid layout -->
button onclick="appendValue('7')" id="btn7">7</span>
button onclick="calculate()" id="equalBtn">= </span>

Step 2: CSS Styling

Use CSS Grid for perfect button alignment

/* Button grid */
.calc-grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  gap: 0.5rem;
}

/* Colored buttons */
.calc-btn {
  background: #dbeafe;
  transition: transform 0.2s;
}

Step 3: JavaScript Logic

Handle button clicks with JavaScript

// Append value to display
function appendValue(value) {
  const display = document.getElementById('display');
  display.value += value;
}

// Calculate result
function calculate() {
  const display = document.getElementById('display');
  try {
    display.value = eval(display.value);
  } catch (e) {
    display.value = 'Error';
  }
}
